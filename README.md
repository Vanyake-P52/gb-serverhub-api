# GB ServerHub API

API для управления пользовательскими игровыми серверами / API for managing user game servers.  
Неофициальный проект, не является продуктом Gorebox или Photon / Unofficial project, not a product of Gorebox or Photon.  
Позволяет хранить коды серверов, добавлять новые и получать список доступных серверов / Allows storing server codes, adding new servers, and retrieving the list of available servers.
